Goal:  My objective is to send Windows failed login alerts(Event ID 4625) from ELKStack to Shuffle using a webhook so Shuffle can run an automated workflow that sends a Discord notification. This helps me review the alert and take the proper action. 


# Step 1: Create the Webhook Trigger in Shuffle
1. Open "Shuffle" 

2. Got to "Workflows"

3. Open my workflow and create a new on

4. Click "Triggers"

5. Select "Webhook"

6. Create the webhook trigger

7. Copy the "Webhook URL" (I will paste this into Kibana)
